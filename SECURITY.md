# Security Policy

## Supported Versions

Currently, only the latest `master` branch is supported with security updates.

## Reporting a Vulnerability

We take the security of `ops-engine` very seriously. If you discover a vulnerability, especially regarding webhook signature validation, rate limiting circumvention, or data exposure, we ask that you report it to us confidentially.

**Please DO NOT open a public issue.**

Instead, please email us at:
**support@langevc.com**

You should receive a response within 48 hours. If the issue is confirmed, we will release a patch as soon as possible.
